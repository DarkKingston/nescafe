Description
This module, based on PHP-IBAN library and the iban_validator module, implements a webform IBAN-field.

Dependencies
- Webform

Usage
- Install with composer $ composer require 'drupal/webform_iban_field:^1.0'
- Create a webform
- Add Element, search for IBAN
